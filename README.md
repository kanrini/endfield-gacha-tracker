# [명일방주: 엔드필드 가챠 분석기]

## [개요]
본 스크립트는 게임 로그를 통해 가챠(헤드헌팅) 기록을 추출하고 확률을 분석하는 도구입니다.
Windows 및 Linux (Steam Proton/Wine) 환경을 모두 지원하며, 외부 라이브러리 설치 없이 Python 내장 모듈만으로 가볍게 동작합니다.
즉, 추가적인 라이브러리 설치가 필요없습니다.

[사용 방법]
1. 시스템에 Python 3.8 이상이 설치되어 있는지 확인합니다. (https://www.python.org)
2. 게임을 실행한 후 게임 내에서 '가챠 기록' 창을 최소 한 번 이상 열어줍니다. (로그 파일에 최신 인증 토큰을 남기기 위함입니다.)
3. 스크립트가 위치한 폴더에서 아래 명령어를 실행합니다:
```bash
python endfield_tracker.py
```
4. 분석이 완료되면 콘솔에 종합 평가가 출력되며, 전체 상세 기록은 동일한 폴더에 CSV 파일로 자동 저장됩니다.

[주의사항]
- 본 프로그램은 비공식 도구입니다.
- 추출된 인증 토큰은 오직 게임의 공식 서버(ef-webview.gryphline.com)에서 데이터를 불러오기 위해서만 전송되며, 어떠한 외부 서버로도 유출되거나 로컬에 저장되지 않습니다.
- 본 도구의 사용으로 인해 발생하는 계정 관련 불이익에 대해서는 개발자가 책임지지 않습니다.
