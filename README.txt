[엔드필드 가챠 분석기]

[사용 방법]
1. Python 3.8 이상 설치 (https://www.python.org)
2. 이 폴더에서 아래 명령어 실행:
   pip install -r requirements.txt
3. 게임 실행 후 가챠 기록 창을 한 번 열기
4. 스크립트 실행:
   python endfield_tracker.py

[주의사항]
- 이 도구는 비공식 도구입니다.
- 인증 토큰은 공식 서버(ef-webview.gryphline.com)에만 전송됩니다.
- 계정 제재 등의 책임은 지지 않습니다.
